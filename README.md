Introduce 
- 메뉴를 고민하는 사람들을 위한 점심 메뉴 랜덤 추첨 웹서비스
- 메뉴를 추천하고 해당 메뉴를 판매하는 식당에 대한 정보 제공
- 방문한 식당에 대해 후기 작성 기능 제공
- 커뮤니티 게시판 기능 제공

개발 환경 및 기술
- Java
- Maven
- myBatis
- HTML / CSS
- Oracle
- Git & Github
